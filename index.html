01

키노라이츠 데이터 크롤링

노트북에 저장된 이전 크롤링 결과와 현재 키노라이츠 페이지를 다시 실행해 얻은 결과를 CSV로 정리하고, 순위 변화와 신규 진입 작품을 비교 그래프로 시각화했습니다.

  • 프로젝트: 키노라이츠 데이터 크롤링
  • 역할: Selenium 기반 자동 수집, 현재 결과 재수집, CSV 비교 및 그래프 시각화
  • 성과: 이전 Top100과 현재 Top20 비교 CSV 및 신규 진입 그래프 생성
  • 결과: 이전 CSV, 현재 CSV, 비교 CSV, 비교 그래프 산출
02

따릉이 이용량 예측 및 거치대 수 증감 판단

서울시 따릉이 이용 데이터와 기상 데이터를 결합해 이용량을 예측하고, 구별 거치대 수를 늘리거나 줄일 필요가 있는지 판단한 데이터 분석 프로젝트입니다.

  • 프로젝트: 서울시 따릉이 이용량 예측 및 거치대 수 증감 판단
  • 역할: EDA, 데이터 병합, 회귀 모델 비교 및 결과 분석
  • 성과: 구별 거치대 증가·유지·감소 판단 기준 도출
03

CCTV 실시간 낙상 감지 시스템

YOLOv5와 OpenCV로 CCTV 화면 속 사람의 자세 변화를 분석하고, 낙상으로 판단되는 상황을 PyQt5 GUI에서 실시간으로 확인할 수 있도록 구현했습니다.

  • 역할: 모델 최적화, 판별 조건 튜닝, GUI 개발
  • 기술: Python, YOLOv5, OpenCV, PyQt5
  • 성과: 실시간 분석 화면, 이벤트 로그, 낙상 알림 기능 구현

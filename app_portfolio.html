01

오초이스 OTT 앱 기능 개발 및 유지보수

국내 최대 케이블 VOD 사업자인 홈초이스의 오초이스 앱에서 콘텐츠 탐색, 재생, 기기 연동을 고려한 OTT 서비스 화면을 개발·유지보수했습니다.

  • 기술: Flutter, Dart, Provider, REST API
  • 서비스 규모: 가입자 수 50,000+, 콘텐츠 제공 수 30,000여 편
  • 핵심 기능: N스크린, 이용자 맞춤 큐레이션, 자막 ON/OFF, 0.25~2배속 재생
  • 지원 기기: Mobile, Tablet, STB, SmartTV
  • 역할: 콘텐츠 UI 구현, API 데이터 렌더링, QA 피드백 반영, iOS·Android 운영 대응

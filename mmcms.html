SEO와의 연결

검색엔진 크롤러가 웹 페이지를 탐색하는 방식 그대로 Selenium을 활용해 동적 페이지를 분석했습니다. robots.txt 구조, 동적 로딩 처리, CSS 선택자와 XPath 활용은 SEO 최적화와 직결되는 경험입니다.

주요 구현

  • HTML 구조 분석: 동적 JavaScript 렌더링 페이지의 HTML 구조를 직접 분석, CSS 선택자·XPath로 콘텐츠 메타데이터 1,000건 이상 추출
  • 무한 스크롤 처리: execute_script 기반 동적 DOM 로딩 대기 시간 조정으로 수집 실패율 대폭 감소
  • 완전 자동화: Render Cron Job으로 하루 1회 자동 실행, Docker 컨테이너화로 환경 일관성 확보
  • 웹 접근성 구조 이해: 웹 접근성 기준의 마크업 구조가 크롤러 수집 효율에 미치는 영향을 직접 경험

성과

  • 수동으로 하던 콘텐츠 데이터 수집 작업을 100% 자동화
  • 검색엔진 크롤러 관점으로 웹 구조를 이해하는 실전 경험 축적

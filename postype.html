Hero Pose Challenge — 실시간 다중 인물 Pose Detection

채용 과제 · Android (Java), TensorFlow Lite, CameraX, MoveNet MultiPose

기술 선택 이유

ML Kit는 단일 인물 중심이라 최대 4명 동시 감지가 어려웠습니다. 다중 인물을 직접 지원하는 MoveNet MultiPose 모델을 선택하고, confidence 기준으로 상위 4명만 렌더링하도록 필터링했습니다.

주요 구현

  • CameraX 프리뷰/분석 분리: 카메라 프리뷰와 TFLite 추론 프레임을 별도 use case로 분리. TFLite 추론과 UI 업데이트를 메인 스레드와 분리해 실시간 10FPS 처리 달성
  • 모델 출력 파싱: 모델 출력 형태를 분석해 사람별 17개 keypoint와 person score를 추출하는 파싱 로직 직접 구현
  • 커스텀 Overlay View: keypoint, joint line, bounding box를 Canvas에 직접 렌더링. 카메라 YUV 프레임 → 모델 입력 → overlay 좌표계 변환 흐름을 각 단계로 분리
  • 디버그 UI: FPS, latency, 감지 인원 수를 실시간으로 표시하는 디버그 오버레이 구현

성과

  • 외부 SDK 없이 TFLite 모델 출력을 직접 파싱하고 커스텀 렌더링까지 구현
  • Android 네이티브(Java)로 ML 추론 + 실시간 카메라 처리 전 과정 경험

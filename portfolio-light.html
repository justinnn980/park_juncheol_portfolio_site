헬로tv → 오초이스 앱 마이그레이션 + 웹 전환

2025.02 ~ 2025.10 (9개월) · Flutter, React, Spring Boot, AWS · 팀 4명

목표

LG헬로비전 헬로tv 앱을 오초이스로 안정적으로 전환하고, 영화 탐색 페이지를 React 웹으로 분리하여 앱 배포 없이 콘텐츠를 빠르게 반영할 수 있는 운영 구조를 구축했습니다.

주요 경험

  • React 웹 분리 + Flutter WebView 연동: 앱 심사 없이 콘텐츠·배너·화면을 즉시 반영할 수 있는 운영 구조 구축
  • Riverpod으로 셋톱박스 페어링 상태(최대 3대) 분리 관리, 실시간 콘텐츠 UI 즉시 반영
  • HTML/JS 경량화, CDN 적용, 스켈레톤 UI, Lazy Loading → FCP 39% 단축 / LCP 10.8% 개선
  • EC2 Proxy에 Elastic IP 적용으로 방화벽 차단 문제 해결, SSL 인증서 적용으로 전체 구간 HTTPS 전환

성과

  • 앱 배포에 의존하던 콘텐츠 변경 구조를 웹 기반으로 전환 → 운영 대응 속도 대폭 향상
  • 성능 지표 기반 최적화로 사용자가 더 빠르게 콘텐츠를 탐색할 수 있는 모바일 경험 제공

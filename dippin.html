자기소개

Flutter 기반 크로스플랫폼 모바일 앱 개발자로, 1년 10개월간 OTT 서비스 'OChoice'의 iOS/Android 앱 개발·운영을 담당했습니다. 매출 1조 LG HelloVision, 매출 3,500억 d'Live와의 대형 마이그레이션 프로젝트에서 DRM 연동, 플레이어 안정화, 성능 최적화(FCP 39% 단축·LCP 10.8% 개선)까지 앱 서비스의 전 과정을 직접 설계하고 구현했습니다. 문제가 발생하면 라이브러리 내부 구조까지 파고드는 꼼꼼함과, 지표를 기반으로 원인을 찾아 개선하는 계획적인 접근 방식으로 서비스 품질을 높여왔습니다.

핵심 역량

  • Flutter 기반 iOS/Android 앱 개발 및 유지보수 (1년 10개월 실무)
  • WebView, DRM(Multi DRM·NCG DRM), Player 연동 및 안정화
  • REST API 연동, 상태관리(Riverpod), 화면 전환 구현
  • FCP/LCP 지표 기반 앱 성능 분석 및 최적화
  • iOS/Android 앱 출시·버전 관리·배포·인증서 이슈 대응
  • Firebase, Google Analytics, AdMob 연동 및 운영
  • AWS EC2, Docker, Render 기반 배포 환경 구성
  • WebSocket 기반 실시간 통신 설계·구현

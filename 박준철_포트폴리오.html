📺 헬로tv → 오초이스 마이그레이션 + 웹 전환

2025.02 ~ 2025.10 (9개월) · Flutter, React, Spring Boot, AWS · 팀 4명

목표

LG헬로비전 헬로tv 앱을 오초이스로 안정적으로 전환하고, 영화 탐색 페이지를 웹으로 분리하여 앱 배포 없이 콘텐츠를 빠르게 반영할 수 있는 구조 구축.

주요 경험

  • React 기반 웹 분리 + WebView 연동 → 앱 심사 없이 콘텐츠 즉시 반영 가능한 운영 구조
  • Riverpod으로 셋톱박스 페어링 상태(최대 3대) 분리 관리, 실시간 콘텐츠 UI 연동
  • HTML/JS 경량화, CDN 적용, 스켈레톤 UI, Lazy Loading → FCP 39% 단축 / LCP 10.8% 개선
  • EC2 Proxy Elastic IP 고정 + HTTPS(SSL) 전환으로 로드밸런싱 환경 방화벽 차단 문제 해결

성과

  • 앱 배포 의존 구조 → 웹 기반 운영으로 전환, 운영 대응 속도 대폭 향상
  • 성능 지표 기반 최적화로 사용자 콘텐츠 탐색 경험 개선
